R for Analytics

Code written for using R by Ajay Ohri. Updated version of companion code in books R for Business Analytics http://www.amazon.com/R-Business-Analytics-A-Ohri/dp/1461443423 and R for Cloud Computing http://www.amazon.com/Cloud-Computing-Approach-Data-Scientists/dp/1493917013.

Based on teaching R online in India.
